Podemos usar a função type() para descobrir a que classe uma variável ou valor pertence. Por exemplo, type(" ") nos diz que esta é uma classe de string. O único atributo neste caso é o valor da string, mas existem vários métodos associados à classe. Vimos o método upper() , que retorna a string em letras maiúsculas, bem como isnumeric() que retorna um booleano nos informando se a string é ou não um número. Você pode usar o dir()função para imprimir todos os atributos e métodos de um objeto. Cada string é uma instância da classe string, tendo os mesmos métodos da classe pai. Como o conteúdo da string é diferente, os métodos retornarão valores diferentes. Você também pode usar a função help() em um objeto, que retornará a documentação da classe correspondente. Isso mostrará todos os métodos da classe, juntamente com os parâmetros que os métodos recebem, os tipos de valores de retorno e uma descrição dos métodos.
